## Generator pattern
New page batches use self-contained Node.js scripts:
1. Define page data arrays
2. Generate HTML via template function
3. `track(url, priority)` → append to sitemap.xml + _redirects SECTION in one pass
4. Run with `node generate-phase6.js`

**Why:** Avoids manual edits to 470+ URL sitemap. Generator tracks only newly created pages (skip existing), preventing duplicate entries.

Currently at SECTION 20. Each new batch adds one section. No catch-all except the last line `/* → /404.html 404`.
