PHASE 5 — PREMIUM VISUAL EXPERIENCE FOR SPANISH AND FRENCH

Objective:
Transform the Spanish and French sections of Paraíso de Aves from text-heavy pages into a premium, trustworthy, image-led experience using our real photographs.

Languages in scope:

Spanish:
/

French:
/fr/

Do not work on Portuguese in this phase.

IMPORTANT

- Do not remove existing SEO content.
- Do not change existing URLs.
- Do not break current routing.
- Do not reduce page word count.
- Preserve the current premium green and gold brand style.
- Use real uploaded photographs only.
- Do not generate fake facility or bird photos.
- Do not display broken image placeholders.
- Every visual improvement must remain fast, responsive and SEO-friendly.
- Spanish and French pages must have equivalent visual structure, but all text, ALT attributes and captions must be properly localized.

==================================================
1. CREATE A REUSABLE MULTILINGUAL MEDIA SYSTEM
==================================================

Build a reusable image gallery and media component that supports:

- Spanish and French content
- Responsive grid
- Masonry layout where suitable
- Mobile-friendly cards
- Lightbox/full-screen image view
- Keyboard navigation
- Swipe navigation on mobile
- Lazy loading
- WebP and AVIF support
- Width and height attributes
- srcset and sizes
- Loading skeletons
- Graceful fallback if an image is missing
- SEO-friendly filenames
- Localized ALT text
- Localized captions
- ImageObject schema where appropriate

Each image record should support:

{
  src,
  slug,
  category,
  width,
  height,
  altEs,
  altFr,
  captionEs,
  captionFr,
  featured,
  species,
  location,
  dateTaken
}

Create one central media data structure so the same real image can be reused correctly across related Spanish and French pages without duplicate hardcoding.

10. IMAGE SEO
==================================================

For every uploaded image:

- rename to a descriptive SEO-friendly slug
- use lowercase
- use hyphens
- avoid spaces
- avoid random camera filenames
- convert to WebP and AVIF where possible
- preserve original backup
- compress without visible quality loss
- include width and height
- add localized ALT text
- add localized caption where appropriate

Filename examples:

Spanish:
